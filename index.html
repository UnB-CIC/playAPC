PlayCB

Introdução

A PlayCB é uma biblioteca educacional de programação voltada para os alunos de Computação Gráfica da UnB que tem como objetivo ilustrar, de maneira gráfica, os conceitos aprendidos durante o curso com aplicações simples. Com ela, é possível criar animações simples e desenhos estáticos.

A biblioteca consiste em um conjunto de funções para criação e manipulação de formas 2D e utilização de imagens. Utiliza a API gráfica OpenGL e a biblioteca GLFW

A PlayCB foi desenvolvida usando fortemente o conceito de Orientação a Objetos, por praticidade de desenvolvimento e facilidade de encapsulamento. Entretanto, a playcb.h entrega ao aluno uma interface amigável com chamadas de funções simples para a criação de todos os objetos, utilizando o paradigma Imperativo.

Avisos

Esta biblioteca está configurada para executar juntamente com a GLFW 2.7 e qualquer versão da OpenGL

Primeiro exemplo

Se a instalação e configuração da máquina estiver correta, o seguinte código deve exibir uma janela com um plano cartesiano desenhado

#include "playcb.h"
int main(){
MostraPlanoCartesiano(10);
AbreJanela(800,600, "Exemplo 1 - Plano Cartesiano");
PintarFundo(255, 255, 255);
Desenha();
return 0;
}
